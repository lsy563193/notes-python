
# 什么是对象？

在 `Python` 中，几乎所有的东西都是对象。

整数是对象：


```python
a = 257
```


```python
type(a)
```




    int




```python
id(a)
```




    53187032L



`b` 和 `a` 是同一个对象：


```python
b = a
```


```python
id(b)
```




    53187032L




```python
c = 258
id(c)
```




    53186960L



函数：


```python
def foo():
    print 'hi'
```


```python
type(foo)
```




    function




```python
id(foo)
```




    63632664L



`type` 函数本身也是对象：


```python
type(type)
```




    type




```python
id(type)
```




    506070640L



只有一些保留的关键词不是对象：


```python
id(if)
```
