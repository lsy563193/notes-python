
# 矩阵

使用 `mat` 方法将 `2` 维数组转化为矩阵：


```python
import numpy as np
a = np.array([[1,2,4],
              [2,5,3], 
              [7,8,9]])
A = np.mat(a)
A
```




    matrix([[1, 2, 4],
            [2, 5, 3],
            [7, 8, 9]])



也可以使用 **Matlab** 的语法传入一个字符串来生成矩阵：


```python
A = np.mat('1,2,4;2,5,3;7,8,9')
A
```




    matrix([[1, 2, 4],
            [2, 5, 3],
            [7, 8, 9]])



利用分块创造新的矩阵：


```python
a = np.array([[ 1, 2],
              [ 3, 4]])
b = np.array([[10,20], 
              [30,40]])

np.bmat('a,b;b,a')
```




    matrix([[ 1,  2, 10, 20],
            [ 3,  4, 30, 40],
            [10, 20,  1,  2],
            [30, 40,  3,  4]])



矩阵与向量的乘法：


```python
x = np.array([[1], [2], [3]])
x
```




    array([[1],
           [2],
           [3]])




```python
A * x
```




    matrix([[17],
            [21],
            [50]])



`A.I` 表示 `A` 矩阵的逆矩阵：


```python
print A * A.I
```
