
# glob 模块：文件模式匹配


```python
import glob
```

`glob` 模块提供了方便的文件模式匹配方法。

例如，找到所有以 `.ipynb` 结尾的文件名：


```python
glob.glob("*.ipynb")
```




    ['11.03 json.ipynb',

     '11.04 glob.ipynb']



`glob` 函数支持三种格式的语法：

- `*` 匹配单个或多个字符
- `?` 匹配任意单个字符
- `[]` 匹配指定范围内的字符，如：[0-9]匹配数字。

假设我们要匹配第 09 节所有的 `.ipynb` 文件：


```python
glob.glob("../09*/*.ipynb")
```
